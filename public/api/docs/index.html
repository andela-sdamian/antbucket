    Introduction

    Welcome to the AntBucket API. You can use our API to access various endpoints which includes (bucketlists and items). The bucketlists end point houses a collection of various items, while the items endpoints belongs to a bucketlist.

    Endpoints data is returned in JSON format, and whenever you make a request to an endpoint that does not exist, we do our best to handle such scenairos by returning a freindly JSON formated error message.

    You can checkout our GitHub repository if you want to contribute or raise an issue.

    Available End Points

    NOTE: The API is currently at version 1, so ensure you prefix each endpoint with “api/v1/” to access them.
    End Point Function
    POST /users: Create a new user
    POST /auth/login: Logs in a user
    DELETE /auth/logout: Logs out a user
    POST /bucketlists: Creates a new bucket list
    GET /bucketlists: Lists all the created bucket lists
    GET /bucketlists/(id): Gets a single bucket list
    PUT /bucketlists/(id): Updates this single bucket list
    DELETE /bucketlists/(id): Deletes this single bucket list
    POST /bucketlists/(id)/items: Creates a new item in bucket list
    PUT /bucketlists/(id)/items/(item_id): Updates a bucket list item
    DELETE /bucketlists/(id)/items/(item_id): Deletes an item in a bucket list

    Authentication

    Creating a User

    HTTP Status: 201 

    curl -X POST -d "email=john@doe.com&password=password" \
https://antbucket.herokuapp.com/api/v1/users

    The above should return JSON data in the format 

    
  {
    "email": "john@doe.com",
    "created_at": "2016-04-29T12:54:49.785Z",
    "updated_at": "2016-04-29T12:54:49.785Z"
  }

    To use our API, you will have to create an account simply by sending a POST request to the appropriate endpoint

    HTTP Request

    POST https://antbucket.herokuapp.com/api/v1/users

    Query Parameters

    Parameter Description
    email a valid email in the format xxx@domain.com
    password a valid string with a minium of 7 characters

    Try it out

    Remember request must be of type POST

    User Login

    HTTP Status: 200 

    curl -X POST -d "email=john@doe.com&password=password" \
https://antbucket.herokuapp.com/api/v1/auth/login

    The above should return JSON data in the format 

      {
    "user_email": "john@doe.com",
    "token": "xkdsjf8329384929xkls02032032zndd39fsk!23"
  }

    HTTP Request

    POST https://antbucket.herokuapp.com/api/v1/auth/login

    After successfull account creation, you should login to verify your account and get a token. This token would be used for subsequent request sent to the server.

    Query Parameters

    Parameter Description
    email a valid email in the format xxx@domain.com
    password a valid string with a minimu of 7 characters

    Try it out

    User Logout

    HTTP Status: 422

    curl -i -X DELETE https://antbucket.herokuapp.com/api/v1/auth/logout \
-H "token: 029kdfsa0932a.xlkdsfiosdfds"

    The above should return JSON data in the format 

      {
    "message": "signed out sucessfully"
  }

    HTTP Request

    DELETE https://antbucket.herokuapp.com/api/v1/auth/logout

    When a user is done, you may want to expire the user session, simply send a delete request to the server

    Headers

    Header Description
    token a valid user token

    Try it out

    Please ensure you pass in a valid user token or you may not be able to end a user session
